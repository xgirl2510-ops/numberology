## [1.0.0] - 2025-01-13

### Added
- Initial release of numerology-ar library
- Core numerology calculations:
  - Life Path Number
  - Expression Number
  - Soul Urge Number
  - Personality Number
  - Birthday Number
  - Maturity Number
  - Balance Number
  - Hidden Passion Number
  - Subconscious Self
  - Karmic Lesson Numbers
  - Pinnacle Numbers (4 cycles)
  - Challenge Numbers (4 cycles)

### Features
- **30+ Language Support** with dynamic Y vowel/consonant detection:
  - Vietnamese (vi)
  - English (en)
  - French (fr)
  - Spanish (es)
  - German (de)
  - Italian (it)
  - Portuguese (pt)
  - Russian (ru)
  - Ukrainian (uk)
  - Polish (pl)
  - Greek (el)
  - Turkish (tr)
  - Arabic (ar)
  - Hindi (hi)
  - Japanese (ja)
  - Korean (ko)
  - Chinese (zh)
  - Thai (th)
  - Tagalog (tl)
  - Indonesian (id)
  - Malay (ms)
  - Swahili (sw)
  - Zulu (zu)
  - Yoruba (yo)
  - Wolof (wo)
  - Hausa (ha)
  - Igbo (ig)
  - Somali (so)
  - Amharic (am)
  - And more...

- **Name Component Reduction Method**: Reduces each name component (First, Middle, Last) separately before summing
- **Master Numbers Support**: Preserves 11, 22, 33 throughout calculations
- **Comprehensive Interpretations**: Detailed meanings for all numbers (1-9, 11, 22, 33)
- **Vietnamese Name Processing**: Automatic diacritics removal and proper Y handling

### Technical

- Type hints throughout the codebase
- Comprehensive test suite
- Optional Google Translate integration
- Zero required dependencies (googletrans is optional)
